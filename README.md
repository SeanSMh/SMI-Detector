# Privacy Policy

Last updated: 2026-03-23

`SMI Detector` is an IntelliJ Platform plugin designed to analyze project code and detect potential SMI-related issues within JetBrains IDEs.

---

## 1. Data Processing Scope

This plugin processes project files locally in order to provide:

- static code inspection
- SMI-related issue detection
- editor highlighting and tool window visualization
- local inspection reports

The plugin may read:

- source code files in the current project
- file paths and file structure
- basic project metadata required for analysis
- VCS change status for changed-file views
- local plugin settings and configuration files

---

## 2. Local Processing Only

All analysis is performed locally on the user's machine.

The plugin does **not upload source code, personal data, project files, or analysis results to any remote server**.

The plugin does **not include telemetry, analytics, advertising SDKs, or background data collection**.

---

## 3. Data Storage

The plugin may store limited data locally on the user's machine, including:

- analysis cache
- plugin UI preferences
- locally generated inspection reports

This data is stored only for plugin functionality and is not shared externally by the plugin.

---

## 4. Network Access

The plugin does **not require network access** for its core functionality.

All inspection and analysis features operate fully offline.

---

## 5. External Commands

The plugin does **not automatically execute external commands**.

If users manually integrate external tools or scripts outside the plugin, any resulting data transfer is initiated by the user and is outside the control of the plugin.

---

## 6. Data Sharing

The plugin does **not sell, transmit, or share user data with third parties**.

---

## 7. User Control

Users may stop data processing by:

- disabling the plugin
- uninstalling the plugin

Users may remove locally stored plugin data by deleting plugin caches and local configuration files from their machine.

---

## 8. Children’s Privacy

This plugin is intended for software development use and is **not directed to children**.

---

## 9. Changes to This Policy

This privacy policy may be updated from time to time.  
Material changes will be reflected in the updated policy text and the "Last updated" date above.

# Privacy Policy

Last updated: 2026-03-23

`Code Radar` is an IntelliJ Platform plugin for local code complexity analysis in JetBrains IDEs.

## 1. Data Processing Scope

This plugin processes project files locally in order to provide:

- code complexity scoring
- hotspot detection
- editor and tool window visualization
- local report export
- prompt generation for refactoring assistance

The plugin may read:

- source code files in the current project
- file paths and file structure
- basic project metadata needed for analysis
- VCS change status for changed-file views
- local plugin settings and configuration files such as `radar.yaml`

## 2. Local Processing Only

All analysis is performed locally on the user's machine.

The plugin does not upload source code, personal data, project files, or analysis results to any remote server.

The plugin does not include telemetry, analytics, advertising SDKs, or background data collection.

## 3. Data Storage

The plugin may store data locally on the user's machine, including:

- analysis cache
- plugin UI preferences
- local exported reports
- locally generated prompt files, if the user chooses to save them

This data is stored only for plugin functionality and is not shared externally by the plugin itself.

## 4. Network Access

The plugin does not require network access for its core functionality.

If the user explicitly copies generated prompts to another tool, exports reports, or configures and runs an external command, any subsequent data transfer is outside the plugin's control and is initiated by the user.

## 5. External Commands

The plugin may allow the user to configure and run external commands manually.

Such commands are executed only when explicitly triggered by the user. The plugin is not responsible for the privacy, security, or behavior of third-party tools or scripts launched by the user.

## 6. Data Sharing

The plugin does not sell, transmit, or share user data with third parties.

## 7. User Control

Users may stop data processing by disabling or uninstalling the plugin.

Users may remove locally stored plugin data by deleting plugin caches, exported files, and local configuration data from their machine.

## 8. Children’s Privacy

This plugin is intended for software development use and is not directed to children.

## 9. Changes to This Policy

This privacy policy may be updated from time to time. Material changes will be reflected in the updated policy text and date above.
